# CHANGELOG

## [v0.0.1] 2022.02.27

- 支持模板在线安装与卸载
- 支持插件在线安装与卸载
- 完成v0.0.1版本在线文档
- 完成api在线文档

## [v0.0.2] 2022.03.28

- 新增全文检索，系统默认接入lucene，可通过插件扩展接入其他的全文检索引擎
- 权限新增部门管理功能
- 新增对外部http请求的封装
- 新增短信sms支持，系统默认接入阿里云短信，可通过插件扩展接入其他厂家的短信服务
- 插件新增支持自定义freemarker标签
- 插件新增支持注册自定义mybatisMapper.xml
- 支付插件业务扩展抽象类结构调整
- 修复一些已知Bug
