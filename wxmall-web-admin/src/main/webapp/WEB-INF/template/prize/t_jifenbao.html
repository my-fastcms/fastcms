集分宝
非系统自动发放。用户兑换后会留下支付宝帐号，您需要手工在系统外为其发放集分宝，然后在系统中登记发放状态。
  • * 显示在兑换列表中，例如：限时1000积分兑换10个集分宝哦
  • *   至   兑换话费活动什么时间开始，什么时间结束
  • 需要的积分： * 需要粉丝多少积分才能兑换一次
  • 发行集分宝总数:  * 本次有效时间内，所有粉丝可以兑换的集分宝发行总数
  • 兑换集分宝个数:  * 本次有效时间内，一个粉丝一次用积分兑换多少个集分宝
  • 兑换次数： * 每个粉丝允许兑换的次数
